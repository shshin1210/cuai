# webtoon-recommendation

## 1. data
* **all-webtoons.csv**: 제목, 장르, 스토리, 썸네일 등 크롤링
  * 네이버 웹툰 (21.07.22 기준)
  * 다음 웹툰 (21.07.25 기준)
  * 카카오 페이지 (21.08.13 기준)
  * 레진코믹스 (21.08.14 기준)
* **선호도.csv**: 구글 Forms를 이용해 **선호장르**와 위의 플랫폼에서 본 **취향에 맞는 웹툰**, **취향에 맞지 않는 웹툰** 조사
  * 104 개의 응답, 303 개의 웹툰
